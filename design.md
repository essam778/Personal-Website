# Portfolio Design Style Guide

## Design Philosophy

### Color Palette
- **Primary Dark**: Deep navy (#0a192f) - Professional, tech-focused background
- **Secondary Dark**: Light navy (#112240) - Card backgrounds and sections
- **Accent**: Vibrant cyan (#64ffda) - Highlights, buttons, and interactive elements
- **Text Primary**: Light slate (#e6f1ff) - Main content text
- **Text Secondary**: Slate (#8892b0) - Secondary text and descriptions
- **White**: Pure white (#ffffff) - Contrast text and highlights

### Typography
- **Display Font**: "Inter" - Modern, clean sans-serif for headings
- **Body Font**: "Inter" - Consistent typography throughout
- **Code Font**: "JetBrains Mono" - For technical snippets and labels
- **Font Hierarchy**:
  - H1: 3.5rem (56px) - Hero name
  - H2: 2.5rem (40px) - Section headers
  - H3: 1.5rem (24px) - Subsection headers
  - Body: 1rem (16px) - Regular text
  - Small: 0.875rem (14px) - Labels and captions

### Visual Language
- **Minimalist Tech Aesthetic**: Clean lines, generous whitespace, surgical precision
- **Cyber-Inspired Elements**: Subtle glow effects, geometric patterns
- **Professional Yet Youthful**: Sophisticated dark theme with vibrant accents
- **Hardware Meets Software**: Visual metaphors combining circuits and code

## Visual Effects

### Used Libraries
- **Anime.js**: Smooth animations and transitions
- **Typed.js**: Typewriter effect for hero text
- **Splitting.js**: Text animation effects
- **ECharts.js**: Skills visualization charts
- **p5.js**: Interactive background particles
- **Pixi.js**: Hardware-accelerated visual effects
- **Matter.js**: Physics-based animations for project showcase

### Background Effects
- **Particle System**: Floating geometric shapes representing circuits and nodes
- **Animated Grid**: Subtle moving grid pattern suggesting precision and structure
- **Shader Effects**: Liquid-metal displacement effects for section transitions

### Animation Strategy
- **Typewriter Animation**: Hero name and tagline with cursor effect
- **Stagger Animations**: Skills cards and project items appear with delays
- **Hover Transformations**: 3D tilt effects on interactive elements
- **Scroll Parallax**: Subtle depth effects on background elements
- **Glow Transitions**: Accent colors pulse on interactive elements

### Interactive Elements
- **Skill Bars**: Animated progression bars with percentage counters
- **Project Cards**: Hover effects with image zoom and overlay information
- **Navigation**: Smooth underline animations for active states
- **Contact Form**: Floating labels with focus animations

### Header Effects
- **Animated Particles**: Circuit-board inspired particles flowing across hero
- **Gradient Text**: Name with animated gradient cycling
- **Floating Elements**: Subtle movement on profile image and text

### Component Styling
- **Cards**: Dark backgrounds with subtle borders and shadow lift on hover
- **Buttons**: Gradient backgrounds with glow effects on hover
- **Form Elements**: Dark inputs with cyan focus states
- **Images**: Subtle border radius with overlay effects

### Responsive Design
- **Mobile-First**: Optimized for touch interactions
- **Breakpoints**: 
  - Mobile: 320px - 768px
  - Tablet: 768px - 1024px
  - Desktop: 1024px+
- **Touch Targets**: Minimum 44px for mobile interactions

### Accessibility Features
- **High Contrast**: 4.5:1 minimum contrast ratio for all text
- **Focus Indicators**: Clear visual focus states for keyboard navigation
- **Screen Reader Support**: Proper ARIA labels and semantic HTML
- **Reduced Motion**: Respect user preferences for motion sensitivity

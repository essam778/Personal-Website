Smart Car Robot

Project Overview

This autonomous smart car represents my first major robotics project, combining hardware design, sensor integration, and intelligent programming. The vehicle can navigate obstacles, follow lines, and be controlled wirelessly via Bluetooth.

Technical Implementation

  • • Microcontroller: Arduino Uno R3 for main control
  • • Motors: Dual DC motors with L298N motor driver
  • • Sensors: HC-SR04 ultrasonic sensor for obstacle detection
  • • Line Detection: IR sensor array for line following
  • • Communication: HC-05 Bluetooth module for wireless control
  • • Power: 18650 Li-ion battery pack with voltage regulation

Features

Autonomous Mode

Obstacle avoidance using ultrasonic sensors

Line Following

Precise line detection and following algorithms

Manual Control

Bluetooth-based smartphone control

Speed Control

PWM-based motor speed regulation

Project Overview

This cutting-edge project combines the power of ESP32 microcontrollers with Google's Gemini AI, creating an intelligent IoT system capable of real-time data processing, natural language understanding, and autonomous decision-making.

  • • Microcontroller: ESP32-WROOM-32 with WiFi and Bluetooth

Voice commands and text-based interactions

Smart Weather Station

Project Overview

This project is an IoT-based weather monitoring system. It collects real-time environmental data such as temperature, humidity, and pressure, stores it in the cloud, and allows users to view this data through a mobile app.

Technical Stack

  • • Microcontroller: ESP32 with Wi-Fi and Bluetooth
  • • Sensors: DHT22 (Temperature/Humidity), BMP180 (Pressure)
  • • Cloud API: Firebase for real-time data storage
  • • Mobile App: Integration with mobile apps for remote monitoring
  • • Communication: MQTT protocol for data transmission
  • • Power Supply: Powered by a 5V DC adapter

Key Features

Real-Time Monitoring

Monitor temperature, humidity, and pressure in real-time through a cloud-based dashboard.

Cloud Integration

Store and access data securely in the cloud using Firebase.

Mobile App

Control and monitor the weather station remotely using a custom mobile app.

Data Alerts

Get notifications for significant changes in environmental conditions (e.g., high temperature).
